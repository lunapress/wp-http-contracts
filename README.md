# WP HTTP Wrapper Contracts

Contracts and wrappers for WordPress HTTP API.

## Supported Functions

- `wp_remote_get()`
- `wp_remote_post()`
- `wp_remote_request()`
- `wp_safe_remote_get()`
- `wp_safe_remote_post()`
- `wp_remote_retrieve_body()`
- `wp_remote_retrieve_response_code()`
- `wp_remote_retrieve_headers()`
